# Design System

Visual language, tokens, and component patterns for ViewMarket.

## Goals

- Consistent monochrome aesthetic (dark and light themes)
- Compact, high-density financial UI
- CSS custom properties for all theming (no Tailwind utilities)
- Accessible, professional look and feel

## Files

- `tokens.md` — Color palette, spacing scale, typography, border styles
- `components.md` — Component inventory, usage guidelines, states
- `patterns.md` — Recurring layouts: lists, modals, forms, tables

## Key Decisions

- Dark base: `#121212`, Light base: `#ffffff`
- Layered surface system using alpha overlays
- Primary text size: `0.875rem`, badges: `0.625rem`
- Full-width list separators, not floating cards
